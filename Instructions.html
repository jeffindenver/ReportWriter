Cellular One Daily Report Step by Step Instructions

Step 1

First, you will want to gather all the necessary files. You may download all files from the FTPS site using your favorite FTP program. Site: ftps.us.caas.hosted-inin.com

Download these required files

Folder: ../ScheduledReports/

  • CellOne_MTD_QBD.xls
  • CellOne_Summary_MTD_QBD.xls
  • CellOne_Summary_WTD_QBD.xls
  • CellOne_WTD_QBD.xls

Folder: ../ScheduledReports/Agents/

  • CellularOne Outbound MTD.xls
  • CellularOne Outbound WTD.xls
  • CellularOne QA MTD.xls
  • ThisMonth_C1.xls
  • ThisWeek_C1.xls

Step 2

Second, the report requires occupancy numbers for the agents. For this, we use the Efficiency Reporter program located at S:\Reports\Utilities\EfficiencyReporter\EfficiencyReporterAuto.jar
  • Double click the jar file to run the program.
  • Drag the files ThisMonth_C1.xls and ThisWeek_C1.xls onto the program window.

The program will process the files and output efficiency reports, which include the occupancy stats. The output file will be written to the same directory from which the files were dragged.

Step 3

Outbound calls are not included in regular agent efficiency reports. Consequently, we need to add them.

  • Open each of the output files.
  • In the agent name column, add rows named "outbound" and "total."
  • Set aside or close these files for now. You'll need them again shortly.

Step 4

Next we need to get outbound minute values for month-to-date (MTD) and week-to-date (WTD) to use with the efficiency reports that were just set aside.

  • Open both files "CellularOne Outbound MTD.xls" and "CellularOne Outbound WTD.xls".
  • Filter the table for outbound calls only.
  • Once filtered, copy the entire Interaction Duration column.
  • Paste the copied column in a new sheet. Highlight a cell in the column and insert a table.
  • In column B next to the new table, add the following formula: =A2*1440
  • Next, either accept the option to use the formula for the length of the column or copy the formula the length of the column manually.
  • Sum column B. The result will be the total number of outbound minutes.
  • Follow these steps for both files.

Step 5

At this point, we should have the total outbound minutes for the current week and the current month. Here we are going to add them to the efficiency reports that were modified earlier.

  • Copy the total minutes and paste the value into each of the following columns: Login Time, Working Time, and Talk Time.
  • Enter Zero in the Available Time column of the outbound row.
  • Select the Login Time, Working Time, and Talk Time cells of the total row then click AutoSum.
  • Select the Available Time and Handle Time cells of the total row then click AutoSum.
  • Extend the column Working Rate and Occupancy.
  • The bottom right cell should now have the number we're looking for.
  • Repeat these steps for both the WTD and MTD efficiency reports.
  • Save both files for later use.

Step 6

One last value must be calculated before we start entering data, and that is the Quality Assurance (QA) score.

  • Open the file, "CellularOne QA MTD.xls"
  • AutoSum the columns, Score and Maximum Achievable.
  • In a nearby cell, add a formula to divide Score by Maximum Achievable.
  • Save the combined QA score.

Step 7

This report is overcomplicated, but we are getting closer to done. In step 7, we enter all the preceding calculations into the report.

  • Open the report file: s:\reports\call centers\Cellular One of NE Arizona\CellOne Daily.xlsx
  • Enter the occupancy and QA score numbers in the appropriate columns.
  • Change the "Report Date" at the top of the sheet to the current date.
  • Select the SubTotals sheet and copy the MTD subtotal ranging from column B to column L.
  • Select the 2019 Data sheet, select the cell for column B and the current month, paste values copied in previous step.
  • Select the "Spend" sheet and enter the spend amount.
  • Manual data entry is complete. Close the file.

Step 8

Next comes the automated data entry. For this, we use the ReportWriter app. Locate the program at S:\Reports\Utilities\ReportWriter\ReportWriter.jar.

  • Make sure the daily report file is closed.
  • Double click the jar file to run the program.
  • Drag the following files on to the program window.
    • CellOne_MTD_QBD
    • CellOne_WTD_QBD
    • CellOne_Summary_MTD_QBD
    • CellOne_Summary_WTD_QBD

The program will process the files and update the tables of the report.

Step 9

The task comes to an end. Lastly, we need to print the summary page (first sheet) as a pdf file and email the pdf to the Cellular One supervisor.

  • Select the first sheet, which is titled "WTD & MTD Summary."
  • Click on the File menu then select Print.
  • Ensure that the printer selected is "Microsoft Print to PDF."
  • Click the large print button.
  • Name the file "CellOneDaily" followed by the date.
  • Email the pdf to the CellOne distribution list.

Mission accomplished. Good job!
